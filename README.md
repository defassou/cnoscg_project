# cnoscg_project
Gestion des Observateurs de la République de Guinée


---------      LES ETAPES A SUIVRE:   ----------

INSTALLE PYTHON SUR TA MACHINE
CLONE LE PROJET: git clone 
ENTRE DANS LE PROJET: cd cnoscg_project

ACTIVE L'ENVIRONNEMENT VIRTUEL: venv\Scripts\active
INSTALLE LES DEPENDANCES: pip install -r requirements.txt
LANCE LE SERVEUR: python manage.py runserver
CLIQUE SUR LE LIEN LOCAL: http://127.0.0.1:8000/
